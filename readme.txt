Social profiles. Facebook, Twitter, LinkedIn, YouTube, Google, Instagramm, Pinterest, Dribble, etc.. (30+ websites)

== Description ==
Social Profiles supports more than 30 social networks and displays it as a floating box on stickied to the left border
All social networks all profiles in one place!
★ Retina Support
★ Full vector icons by @font-face MojoSocial
★ Support for more than 30 services and their brand colors
★ Icon font use private unicode spaces for accessibility
★ Works splendidly on any browser supporting @font-face


== Installation ==

Upload the Social Profiles plugin to your blog, Activate it, then enter your social ids

1, 2, 3: You're done!
